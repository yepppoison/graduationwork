Тема: "Разработка автоматизированной системы анализы заболеваний за сезон (на примере МИАЦ Брянской области)".
Разрабатывается на формах, работа с графиками.

Пользователи от учреждения за период (7 дней) должны предоставлять данные по общему кол-ву пациентов и кол-ву заболевших.
Так же они заполняют информацию о том, сколько человек и с чем диагнозированы(кол-во заболевших разбитое на отдельные заболевания).

Администратор (пользователь от МИАЦ) работает только с представлениями и не может редактировать общие данные, только работать с статистикой,
на основании этой статистики он может выявлять вспышки заболеваний за период по районам и отдельным участкам этих районов.

Гл.администратор (пользователь от МИАЦ) МОЖЕТ изменять уже внесенные данные от учреждений в случае ошибки при заполнении (не работает с данными, имеет доступ на изменение только по запросу учреждения),
учреждение внутри программы может оставить запрос на изменение данных.
Если гл.администратор вносит изменения без запроса и мы видим это по отчетам/логам (он не имеет к ним доступ), карается администрацией и 02.

Зам.директора по информационной работе со статистикой (пользовать от МИАЦ) имеет полный доступ, так же отслеживает действия гл.администратора,
не имеет возможности удалять отчеты/логи.

Директор с карт-бланшом на свинство

Так же запрос от учреждения на изменения статистики за период отправляется только директором(гл.врачом) учреждения для усиления безопасности системы.
